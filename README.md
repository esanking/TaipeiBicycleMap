# YouBike地圖網站 [TaipeiBicycleMap](https://esanking.github.io/TaipeiBicycleMap/)

### 功能
網頁功能：
* select 選單選擇區域

##
### 使用技術
* 使用 leaflet.js 與 OpenStreetMap 搭建地圖資料
* Vue.js
* Vue CLI 4
* RWD 響應式畫面
* jQuery 互動效果
* ES6
* Webpack
* ESlint(aribnb) 程式碼管理
* Bootstrap 4 及 SCSS 的應用
* 第三方 Plugin 的運用
  * leaflet
  * axios
  * vue-axios
  * jQuery
  * popper.js
  * Bootstrap 4
